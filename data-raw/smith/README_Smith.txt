Package content:
README_Smith							Data version, licensing information and list of brain regions, including for each region: region number, digital atlas label, the corresponding original region
'label' folder							Containing individual manually drawn labels, used in atlas digitisation
lh.atlas.gcs & rh.atlas.gcs					Surface-based atlas
lh.fsaverage.Smith.label.gii & rh.fsaverage.Smith.label.gii 	Surface-based atlas, applied to fsaverage and converted to .gii surface mesh using mris_convert 
Smith.nii							Colin27 volume-based atlas
Smith_NCBI152.nii.gz						NCBI152 volume-based atlas
lh.colortable.txt & rh.colortable.txt				Colortable files for the surface-based atlas
SmithColorLUT.txt						Lookup table, including ROI number and color code for each region
